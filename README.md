 Machine Learning Algorithms Repository

This repository contains implementations of fundamental **Machine Learning algorithms** organized by category.  
It is designed for learning, experimentation, and building a strong foundation in ML concepts.



---

## 📌 Algorithms Included

### 🔹 Supervised Learning
- **KNN (K-Nearest Neighbors)**  
  Simple distance-based classification algorithm.

- **Decision Tree**  
  Tree-based model used for classification and regression.

- **Random Forest**  
  Ensemble of decision trees to improve accuracy and reduce overfitting.

- **SVM (Support Vector Machine)**  
  Finds optimal hyperplane for classification tasks.

- **Naive Bayes**  
  Probabilistic classifier based on Bayes' theorem.

---

### 🔹 Ensemble Learning
- **Boosting**  
  Combines weak learners to create a strong predictive model (e.g., AdaBoost, Gradient Boosting).

---

### 🔹 Unsupervised Learning
- **Clustering**  
  Groups similar data points together (e.g., K-Means, Hierarchical Clustering, DBSCAN).
